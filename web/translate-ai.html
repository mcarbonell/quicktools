What is AI Translation?

AI Translation uses Chrome's local AI (Translator API + Language Detector) or Google Gemini to translate text between languages with high accuracy. Supports 100+ languages with automatic language detection.

How to translate with AI?

  1. Paste or type your text in the source area
  2. Click 'Detect language' or select source language
  3. Choose target language from the dropdown
  4. Click 'Translate' to get instant translation

Use cases for AI translation

  • Translate documents and articles
  • Communicate in foreign languages
  • Localize website content
  • Understand international emails
  • Learn new languages with context
