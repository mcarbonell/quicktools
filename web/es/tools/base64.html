Codificar/Decodificar Base64

Codifica y decodifica texto en Base64.

¿Qué es Base64?
Base64 es un sistema de codificación que convierte datos binarios en texto ASCII utilizando 64 caracteres diferentes. Es comúnmente usado para:
• Enviar datos binarios por email
• Codificar imágenes en línea en HTML/CSS
• Transmitir datos que podrían contener caracteres especiales
• Almacenar datos binarios en formatos de texto

¿Qué es la codificación Base64?

Base64 es un sistema de codificación que convierte datos binarios en texto ASCII utilizando 64 caracteres diferentes (A-Z, a-z, 0-9, +, /). Es ampliamente usado en desarrollo web, sistemas de email y transmisión de datos. Aprende más sobre Base64 en Wikipedia.

¿Cómo usar el codificador/decodificador Base64?

  1. Pega o escribe tu texto en el área de entrada
  2. Haz clic en 'Codificar en Base64' para convertir texto a formato Base64
  3. Usa 'Decodificar Base64' para convertir Base64 de vuelta a texto legible
  4. Copia el resultado usando el botón de copiar

Casos de uso comunes de Base64

  • Codificar imágenes en CSS/HTML usando data URIs
  • Enviar datos binarios en formatos JSON o XML
  • Transmitir archivos adjuntos de email (codificación MIME)
  • Almacenar credenciales o tokens en archivos de configuración
  • Ofuscar texto (no es encriptación, solo codificación)
