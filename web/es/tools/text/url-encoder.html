Codificar/Decodificar URL

Codifica y decodifica URLs y parámetros. Funciona offline en tu navegador.

Diferencias entre codificar URL y componente:
• encodeURI: Codifica una URL completa preservando caracteres especiales como / ? : @ & = + $ #
• encodeURIComponent: Codifica un componente de URL (como un parámetro), escapando más caracteres
